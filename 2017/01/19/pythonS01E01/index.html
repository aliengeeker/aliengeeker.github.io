print()函数的用法:

print函数看起来简单,但是其实要用的灵活还是有很多技巧的.

python print也支持参数格式化

1.格式化输出整数

%s字符串

2.格式化输出16制整数

%x — hex 十六进制（字母部分使用小写）

%X — hex 十六进制（字母部分使用大写）

%d — dec 十进制

%o — oct 八进制

3.格式化输出浮点数(float)

%f浮点数

浮点数默认显示6位小数，使用科学计数法显示浮点数
%e,小写的e
%E,大写的E

4.格式化输出字符串(string)

5.输出列表(list)

6输出出字典(dictionary)

总结一下print()函数的格式化输出：
有人说“java在很多方面就像是C++”，其实我感觉，python3的print函数非常类似于C语言的printf函数，特别是格式化输出和流。

对浮点数：print(“%【对齐方式】【字符宽度】【.精度】”%【对应参数】)

对字符串：print(“%【对齐方式】【显示字符宽度】【.显示的字符数】”%【对应参数】)

格式化输出可以参考C语言的printf函数、以及python的format()方法。

print()函数可以接受任意数量的位置参数，还有3个关键字参数，sep、end、file。格式如下：

print(value, …, sep=’ ‘, end=’\n’, file=sys.stdout)

sep的默认值是空格，只有在给了2个或更多的位置参数时才起作用，打印室在参数之间因空格分隔；

end的默认值是\n，所以调用print()函数默认自动换行；

file的默认值是sys.stdout，标准输出流，默认为控制台。

print (‘Hello World’)等同于sys.stdout.write(‘Hello World’)
